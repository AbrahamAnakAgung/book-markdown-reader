# Simple Markdown Book Reader

## How to use
Clone this repo to your computer and run `yarn`

Run the server first:
```bash
cd server
cd api-server
node server
```

Run the front-end:
```bash
// From the root directory
yarn start
```

Made with **React** and **Typescript**
